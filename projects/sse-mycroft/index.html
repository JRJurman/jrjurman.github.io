SSE-Mycroft

This RapDev Project was a 3 week long endeavor to build a modular network system to automate functions in our lab. Several team members worked on the core server, while myself and a group of other students worked on seperate clients that would drive the features of the system.

The core server was written in C#, and the client-apps used anything from ruby, python, javascript, and C#. While the server was responsible for connecting to the different apps, the apps themselves could do anything from parsing speech-to-text, to getting the weather! The core server and all the apps can be found at this github page

SSE Members continue to work on the project to make it even bigger and better!
