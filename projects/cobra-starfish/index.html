Cobra StarFish

This Project was built as a radio automation program for my High School Radio Station, WIRQ (of which I was an active member and DJ). It was a sucessor to another project called Nightcrawler (whose source code has sadly been lost). Nightcrawler and Cobra StarFish were my first big projects, and allowed the station to play music overnight. The application is a simple python script which runs in a terminal session and creates logs after every song plays. [GitHub Page]
