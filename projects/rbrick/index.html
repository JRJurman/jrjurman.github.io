rBrick

rBrick was a research endevor by RIT’s Software Engineering Department, to build an application that allowed people to build Lego Mindstorms programs using special blocks that could be placed on a tablet. The project explored concepts in both Programmability, Natural User Interfaces, and Accessibility. The application integrated with a previous RIT project called jBricks, which was a more complete IDE for the Lego Mindstorms programming language. For rBrick, I was the intial developer, researcher, and project lead. [Google Code Page]
